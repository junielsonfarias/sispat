# SISPAT - Sistema de Gestão Patrimonial

## Visão Geral
O SISPAT é um sistema completo e independente de gestão patrimonial para municípios e organizações públicas. O sistema permite controle total de bens, imóveis e equipamentos, com hierarquia de permissões, gestão municipal com isolamento de dados, dashboards inteligentes, controle de transferências, inventários, etiquetas personalizadas e relatórios customizáveis.

## Funcionalidades Principais

### Sistema de Autenticação e Usuários
- Autenticação JWT com diferentes níveis de permissão
- Hierarquia: superuser, supervisor, admin, usuario, visualizador
- Gestão de perfis e senhas
- Controle de acesso baseado em município
- Sistema de notificações

### Gestão de Patrimônio
- Cadastro completo de bens patrimoniais
- Categorização e classificação por tipos
- Controle de estado, valor e depreciação
- Histórico completo de alterações
- Upload e gestão de documentos
- Sistema de numeração personalizada
- Geração e impressão de etiquetas

### Gestão de Imóveis
- Cadastro completo de imóveis
- Sistema de geolocalização com mapas
- Controle de manutenções e tarefas
- Campos personalizáveis para imóveis
- Relatórios específicos para imóveis
- Templates de relatório customizáveis

### Sistema de Inventários
- Criação e gestão de inventários
- Contagem e verificação de bens
- Relatórios de divergências
- Exportação de dados
- Controle de responsáveis

### Controle de Transferências
- Solicitação de transferências entre setores
- Sistema de aprovação/rejeição
- Controle de empréstimos
- Geração de termos de responsabilidade
- Histórico completo de movimentações

### Dashboards e Relatórios
- Dashboards específicos por tipo de usuário
- Widgets configuráveis (estatísticas, gráficos, atividades)
- Sistema de templates de relatório
- Exportação em PDF e Excel
- Filtros avançados e buscas
- Análises temporais e por setor

### Gestão Administrativa
- Gestão de municípios
- Configuração de setores e locais
- Personalização de temas e layouts
- Configurações de sistema
- Logs de auditoria
- Backup e restore

## Experiência do Usuário

### Personas
1. **Superuser**: Administrador geral do sistema
2. **Supervisor**: Gestor municipal
3. **Admin**: Administrador local
4. **Usuario**: Operador do sistema
5. **Visualizador**: Consulta apenas

### Fluxos Principais
- Login e navegação por dashboards específicos
- Cadastro e edição de patrimônios/imóveis
- Processo de transferência com aprovações
- Criação e execução de inventários
- Geração de relatórios e etiquetas
- Consulta pública de patrimônios

### UI/UX
- Interface responsiva com Tailwind CSS
- Componentes modernos com Shadcn UI
- Design intuitivo e acessível
- Navegação contextual por permissões
- Feedback visual para todas as ações

## Arquitetura Técnica

### Frontend
- React 19.1.1 com TypeScript
- Vite para build e desenvolvimento
- React Router para navegação
- 15+ contextos para gerenciamento de estado
- React Hook Form + Zod para formulários
- Recharts para visualizações

### Backend
- Node.js com Express.js
- PostgreSQL como banco principal
- 80+ endpoints de API RESTful
- Middleware de autenticação JWT
- Sistema de upload de arquivos
- Logs de auditoria completos

### Banco de Dados
- 19 tabelas principais
- Relacionamentos complexos
- Isolamento por município
- Índices otimizados
- Sistema de backup automático

### Infraestrutura
- Servidor único (full-stack)
- Upload local de arquivos
- Sistema de logs estruturados
- Configuração via variáveis de ambiente
