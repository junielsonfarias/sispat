## 🎯 Introdução

Bem-vindo ao **SISPAT (Sistema de Patrimônio)**! Este manual foi criado para ajudá-lo a utilizar
todas as funcionalidades do sistema de forma eficiente e produtiva.

### 🎯 O que é o SISPAT?

O SISPAT é um sistema completo para gestão de patrimônio público, desenvolvido especificamente para
municípios brasileiros. Com ele, você pode:

- 📦 **Cadastrar e gerenciar patrimônios**
- 📊 **Gerar relatórios detalhados**
- 🏷️ **Criar etiquetas personalizadas**
- 👥 **Gerenciar usuários e permissões**
- 📈 **Acompanhar estatísticas em tempo real**

### 🎨 Interface do Sistema

O SISPAT possui uma interface moderna e intuitiva, com:

- **Design responsivo**: Funciona em desktop, tablet e mobile
- **Navegação simples**: Menu lateral com acesso rápido
- **Dashboard personalizado**: Informações relevantes em tempo real
- **Tema claro/escuro**: Escolha o que for mais confortável

## 🔑 Primeiro Acesso

### 1. Acessando o Sistema

1. **Abra seu navegador** (Chrome, Firefox, Safari, Edge)
2. **Digite o endereço** fornecido pelo administrador
3. **Aguarde o carregamento** da página de login

### 2. Fazendo Login

1. **Digite seu usuário** (fornecido pelo administrador)
2. **Digite sua senha** (fornecida pelo administrador)
3. **Clique em "Entrar"**

### 3. Primeira Configuração

Na primeira vez que acessar:

1. **Altere sua senha** (obrigatório)
2. **Complete seu perfil** com informações adicionais
3. **Configure suas preferências**

## 📦 Gerenciamento de Patrimônios

### Cadastrar Novo Patrimônio

1. **Clique em "Patrimônios"** no menu lateral
2. **Clique em "Novo Patrimônio"**
3. **Preencha os dados obrigatórios**:
   - **Número**: Número único do patrimônio
   - **Descrição**: Descrição detalhada do bem
   - **Categoria**: Categoria do patrimônio
   - **Setor**: Setor responsável
4. **Preencha dados opcionais**:
   - **Marca/Modelo**: Marca e modelo
   - **Número de Série**: Número de série
   - **Valor**: Valor de aquisição
   - **Data de Aquisição**: Data de compra
   - **Localização**: Local físico
5. **Clique em "Salvar"**

### Editar Patrimônio

1. **Acesse a lista de patrimônios**
2. **Clique no patrimônio desejado**
3. **Clique em "Editar"**
4. **Modifique os dados necessários**
5. **Clique em "Salvar"**

### Buscar Patrimônios

#### Busca Simples

1. **Digite o termo** na caixa de busca
2. **Pressione Enter** ou clique na lupa
3. **Visualize os resultados**

#### Busca Avançada

1. **Clique em "Filtros Avançados"**
2. **Selecione os critérios**:
   - **Setor**: Setor específico
   - **Categoria**: Categoria específica
   - **Status**: Ativo, inativo, baixado
   - **Período**: Data de aquisição
   - **Valor**: Faixa de valor
3. **Clique em "Buscar"**

### Transferir Patrimônio

1. **Acesse o patrimônio** desejado
2. **Clique em "Transferir"**
3. **Selecione o novo setor**
4. **Adicione observações** (opcional)
5. **Clique em "Confirmar Transferência"**

### Baixar Patrimônio

1. **Acesse o patrimônio** desejado
2. **Clique em "Baixar"**
3. **Selecione o motivo da baixa**:
   - **Venda**: Patrimônio vendido
   - **Doação**: Patrimônio doado
   - **Inutilização**: Patrimônio inutilizado
   - **Furto/Roubo**: Patrimônio furtado/roubado
4. **Adicione observações**
5. **Clique em "Confirmar Baixa"**

## 📊 Relatórios

### Tipos de Relatórios Disponíveis

1. **Relatório de Patrimônios por Setor**
2. **Relatório de Patrimônios por Categoria**
3. **Relatório de Transferências**
4. **Relatório de Baixas**
5. **Relatório de Inventário**
6. **Relatório de Valor Total**

### Gerar Relatório

1. **Clique em "Relatórios"** no menu lateral
2. **Selecione o tipo de relatório**
3. **Configure os filtros**:
   - **Período**: Data inicial e final
   - **Setor**: Setores específicos
   - **Categoria**: Categorias específicas
   - **Status**: Status dos patrimônios
4. **Clique em "Gerar Relatório"**
5. **Aguarde o processamento**
6. **Visualize ou baixe o relatório**

### Exportar Relatórios

Os relatórios podem ser exportados em:

- **PDF**: Para impressão
- **Excel**: Para análise de dados
- **CSV**: Para importação em outros sistemas

## 🏷️ Etiquetas

### Gerar Etiquetas

1. **Acesse a lista de patrimônios**
2. **Selecione os patrimônios** desejados
3. **Clique em "Gerar Etiquetas"**
4. **Escolha o modelo de etiqueta**:
   - **Modelo Padrão**: Etiqueta básica
   - **Modelo Detalhado**: Etiqueta com mais informações
   - **Modelo QR Code**: Etiqueta com QR code
5. **Clique em "Gerar"**
6. **Visualize a prévia**
7. **Clique em "Imprimir"**

### Modelos de Etiquetas

#### Modelo Padrão

- Número do patrimônio
- Descrição
- Setor
- QR Code

#### Modelo Detalhado

- Número do patrimônio
- Descrição completa
- Marca/Modelo
- Setor
- Localização
- QR Code

#### Modelo QR Code

- QR Code grande
- Número do patrimônio
- Descrição resumida

### Configurar Impressão

1. **Acesse as configurações de impressão**
2. **Configure**:
   - **Papel**: Tamanho do papel
   - **Orientação**: Retrato ou paisagem
   - **Margens**: Margens da página
   - **Qualidade**: Qualidade de impressão
3. **Salve as configurações**

## 👤 Perfil e Configurações

### Editar Perfil

1. **Clique no seu nome** no canto superior direito
2. **Selecione "Perfil"**
3. **Edite as informações**:
   - **Nome**: Nome completo
   - **Email**: Email de contato
   - **Telefone**: Telefone de contato
   - **Foto**: Foto do perfil
4. **Clique em "Salvar"**

### Alterar Senha

1. **Acesse seu perfil**
2. **Clique em "Alterar Senha"**
3. **Digite a senha atual**
4. **Digite a nova senha**
5. **Confirme a nova senha**
6. **Clique em "Alterar"**

### Configurações do Sistema

1. **Clique no seu nome** no canto superior direito
2. **Selecione "Configurações"**
3. **Configure**:
   - **Idioma**: Português (padrão)
   - **Tema**: Claro ou escuro
   - **Notificações**: Tipos de notificação
   - **Fuso Horário**: Fuso horário local
4. **Clique em "Salvar"**

### Atalhos de Teclado

- **Ctrl + N**: Novo patrimônio
- **Ctrl + S**: Salvar
- **Ctrl + F**: Buscar
- **Ctrl + P**: Imprimir
- **Esc**: Cancelar/Fechar

### Navegação Rápida

- **Use o menu lateral** para navegação rápida
- **Use a busca** para encontrar patrimônios rapidamente

## ❓ Perguntas Frequentes

### Como cadastrar um patrimônio?

1. Acesse "Patrimônios" → "Novo"
2. Preencha os dados obrigatórios
3. Clique em "Salvar"

### Como gerar um relatório?

1. Acesse "Relatórios"
2. Selecione o tipo de relatório
3. Configure os filtros
4. Clique em "Gerar"

### Como imprimir etiquetas?

1. Selecione os patrimônios
2. Clique em "Gerar Etiquetas"
3. Escolha o modelo
4. Clique em "Imprimir"

### Como transferir um patrimônio?

1. Acesse o patrimônio
2. Clique em "Transferir"
3. Selecione o novo setor
4. Confirme a transferência

### Como alterar minha senha?

1. Acesse seu perfil
2. Clique em "Alterar Senha"
3. Digite a senha atual e nova
4. Confirme a alteração

### O sistema funciona no celular?

Sim! O SISPAT é responsivo e funciona em:

- **Smartphones**: Android e iOS
- **Tablets**: Android e iPad
- **Navegadores**: Chrome, Safari, Firefox

### Como buscar patrimônios?

- **Busca simples**: Digite na caixa de busca
- **Busca avançada**: Use os filtros
- **Busca por número**: Digite o número exato
- **Busca por descrição**: Digite palavras-chave

### Como exportar dados?

1. Gere um relatório
2. Clique em "Exportar"
3. Escolha o formato (PDF, Excel, CSV)
4. Baixe o arquivo
